사주담 (SajuDam) — 진행 기록 / 검증 근거
==========================================

상태: 전체 스토리 US-001~US-010, US-012~US-020(v2) 인수조건 개별 검증 완료 (증거 기반)
테스트: pytest 82/82 PASS (engine 12, interpret 4, web 20, products 19, daily 13, counsel 14)
실행: python app.py → 200 (기본 5000, 데모 확인은 PORT=5001 사용, 5000은 기존 크롤러 앱 점유)

핵심 구현
---------
- saju/calendar_util.py: 음양력(korean_lunar_calendar), 절기=태양황경(ephem),
  일주=JDN 기반 (day_index=(JDN+49)%60) — 라이브러리 대조로 오프셋 49 보정.
- saju/engine.py: 년주(입춘 기준, 1984=갑자 앵커), 월주(황경 구간+오호둔),
  일주(JDN), 시주(오서둔, ((h+1)//2)%12), 오행 집계.
- saju/interpret.py: 일간별 성격 + 오행 과다/부족 + 직업/재물/연애 규칙 해설.
- app.py 팩토리 + auth.py(회원) + payment.py(토스 샌드박스) + web_utils.py.
- 템플릿: base 상속, design.png(사주담) 팔레트. static/css/style.css 디자인 시스템.
